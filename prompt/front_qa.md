프론트엔드 리뷰 페이지 UI/UX 품질 점검 프롬프트
당신은 시니어 프론트엔드 엔지니어이자 프로덕트 디자이너입니다.
첨부된 리뷰 페이지의 화면을 분석하여 UI/UX 완성도를 평가하고 개선 사항을 제안해주세요.
다음 관점에서 매우 엄격하게 검토해주세요.
1. 레이아웃 구조

섹션 간 간격(Vertical Spacing)이 일관적인가
좌우 패딩이 적절한가
콘텐츠 폭(max-width)이 적절한가
화면이 답답하거나 비어 보이는 영역은 없는가
시선 흐름이 자연스러운가
2. 마진/패딩 시스템
아래 기준으로 평가해주세요.

8pt Grid System 준수 여부
섹션 간 spacing 일관성
카드 내부 padding 적절성
제목과 본문 사이 간격
버튼과 콘텐츠 사이 간격
모바일과 데스크톱에서 spacing 차이
문제가 있다면 구체적으로 수치(px)까지 제안해주세요.

현재: 12px
권장: 24px
